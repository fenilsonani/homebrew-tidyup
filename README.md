# Homebrew Tap for TidyUp

Fast system cleanup tool with intelligent caching.

## Installation

```bash
brew tap fenilsonani/tidyup
brew install tidyup
```

Or directly:

```bash
brew install fenilsonani/tidyup/tidyup
```

## Usage

```bash
tidyup scan          # Scan for cleanable files
tidyup dev           # Find dev artifacts (node_modules, etc.)
tidyup large         # Find large files
tidyup clean         # Clean based on config
```
